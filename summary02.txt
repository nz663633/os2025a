< Go 명령어 >

go mod init 프로젝트폴더명 : 모듈 초기화(생성) -> 현재 폴더에 go.mod 파일 생성
go run hello.go : hello.go 파일 실행
go build hello.go : 디렉터리를 빌드(hello.exe라는 실행파일 생성)
